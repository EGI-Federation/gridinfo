Usage:
       glite-info-static-create.sh <module> [-i <interface>]
       
Parameters:
       <module> = the module you are using. E.g.: site
       <interface> = the interface you want to use. E.g.: glue, wlcg (default)
       
Examples:
       glite-info-static-create.sh site
       glite-info-static-create.sh site -i glue

Web site: http://cern.ch/gridinfo
